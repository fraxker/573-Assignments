Dependencies:

In order to run this code, you must have:
* Python 3.10, this version or greater is required due to the usage of the match statement
* Rawsocketpy, this can be installed via that command "pip3 install rawsocketpy"

Compilation and Execution:
In order to run the code, run the command "python3 project3.py" which will then execute the program for 30 seconds and on exit generate the csv file.
